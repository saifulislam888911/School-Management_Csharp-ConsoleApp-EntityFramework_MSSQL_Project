# Migrations Command :

	i. Adding / Creating Migrations : (Creating : Database, Tables, Adding/Removing Columns)
	
		- dotnet ef migrations add "Add Student Table" --project SchoolManagement --context ApplicationDbContext

		- dotnet ef database update --project SchoolManagement --context ApplicationDbContext


	ii. Deleting / Reverting Migrations : (Roll back to a migration, and bellow it - all migrations will be automatically deleted)

		- dotnet ef database update AddedAddressInStudentTable --project SchoolManagement --context ApplicationDbContext
			(Database theke migrations history & Tables delete hobe : Database ae AddedAddressInStudentTable command err pore jei koyta migration selo, will be removed From Database.)

		- dotnet ef migrations remove --project SchoolManagement --context ApplicationDbContext
			(Visual Studio theke migrations file delete hobe : AddedAddressInStudentTable etar pore jei koyta migration selo, thik shey koyber ei command ta run korte hobe.
			 Tahole serial wise nich theke uporer dike migrations gulo remove hote tahkbe from project file, 
			 AddedAddressInStudentTable command taar agg porjonto.)
